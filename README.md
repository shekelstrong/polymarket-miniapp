# Buildo Headshots Mini App

> AI-аватары для резюме, соцсетей и LinkedIn за 30 секунд

Telegram Mini App (открывается через Telegram-бот).

## Локальная разработка

```bash
npm install
npm run dev
# → http://localhost:5173
```

## Build

```bash
npm run build
# → dist/ (committed to repo for Layero static delivery)
```

## Деплой

Платформа: **Layero** (static, dist/ precommitted).

## Стек

- Vite 6 + React 19
- TypeScript 5.4
- Tailwind CSS 3.4
- Brand color: `#7C3AED`
